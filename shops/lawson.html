ローソンで使える決済

💰 電子マネー

  • WAON / 楽天Edy / iD / QUICPay / 独自の電子マネー
  • ※セブングループのnanacoは使用不可

🚃 交通系電子マネー

  • Suica / PASMO / manaca / SUGOCA / TOICA / nimoca / Kitaca / SAPICA / ICOCA / PiTaPa / はやかけん
  • チャージも可能（ローソン銀行ATM または レジ）
  • ※PiTaPaは近畿圏・三重・愛知の店舗で利用可能

💳 クレジットカード

  • VISA / Mastercard / JCB / AMEX / Diners
  • ナチュラルローソン・ローソンストア100・ローソンマートでも利用可
  • 支払い時に「カード払い」と伝えて、自分で操作
  • 10,000円以下ならサインレス

🌟 おすすめクレジットカード

  • 三井住友カード各種（Visaタッチ・Mastercardコンタクトレス対応）→ 最大7％還元（※一部対象外店舗あり）
  • ローソンPontaプラス → 4％還元
  • 三菱UFJカード → 7％還元＋α

💳 ブランドプリペイド

  • au PAY プリペイドカード
  • ソフトバンクカード
  • dカード プリペイド
  • その他の国際ブランド付きプリペイドカード

📡 タッチ決済（コンタクトレス決済）

  • Visaのタッチ決済
  • Mastercard®コンタクトレス
  • JCBのタッチ決済
  • American Expressのタッチ決済
  • ※「クレジットで」と伝えてタッチすればOK

📱 QRコード決済（バーコード決済）

  • PayPay / 楽天ペイ / メルペイ / d払い / au PAY
  • pring / J-Coin Pay / 銀行Pay / ゆうちょPay
  • Alipay / WeChat Pay / クオカード・ペイ / イオンペイ / Smart Code / COIN+
  • ※au PAY・d払いはポイントカードの同時読み取りに対応

🏦 デビットカード

  • 使える（VISAデビット / JCBデビットなど）

🎁 商品券・ギフトカード

  • ダイエー商品券
  • クオカード
  • ※ローソンストア100・ローソンマートでは使用不可

※掲載情報は変更される場合があります。最新の対応状況は店舗でご確認ください。
