サイゼリヤで使える決済

🚃 交通系電子マネー

  • Suica / PASMO / manaca / SUGOCA / TOICA / nimoca / Kitaca / ICOCA / はやかけん
  • ※ほとんどの店舗で使える（ショッピングセンター内の店舗など例外あり）
  • ※対応状況は店舗ごとに異なる → 店舗検索はこちら

💳 クレジットカード

  • VISA / Mastercard / JCB / AMEX / Diners
  • ※2022年6月からJCB・AMEX・Dinersも対応

🌟 おすすめクレジットカード

  • 三井住友カード各種 → Visaのタッチ決済・Mastercardコンタクトレスで最大7％還元（※一部商業施設内の店舗など例外あり）

💳 ブランドプリペイド

  • au PAY プリペイドカード
  • ソフトバンクカード
  • dカード プリペイド

📡 タッチ決済（コンタクトレス決済）

  • Visaのタッチ決済
  • Mastercard®コンタクトレス
  • JCBのタッチ決済
  • American Expressのタッチ決済
  • Diners Club コンタクトレス
  • ※「クレジットカードで」と伝えてタッチすればOK
  • ※9割以上の店舗で利用可能

📱 QRコード決済（バーコード決済）

  • 利用不可（PayPayなどのコード決済は導入されていません）

🏦 デビットカード

  • 使える（VISAデビット / JCBデビットなど）

※掲載情報は変更される場合があります。最新の対応状況は店舗でご確認ください。
